文件清单

本模版的最新版本可以从 https://github.com/wx672/texmf/tree/master/doc/latex/swfu/swfcthesis/tutorial 下载。 模版目录中包含如下内容：

  • swfcthesis.cls — 是论文模版的class文件，它决定了论文的排版格式；
  • readme.html — 本说明文件。
  • tutorial — 小教程，同时也是一个标准模板。其中，
    • tutorial.tex — 源文件；
    • tutorial.pdf — 编译好的PDF；
    • tutorial.bib — 参考文献；
    • tutorial.mkv — 20分钟视频教程；
  • sample — 完整的论文实例；
    • sample.tex — TeX源文件；
    • sample.pdf — 编译完成的PDF文件；
    • sample.bib — 参考文献实例；
    • chapters — 章节实例。此目录中的每个TeX文件对应论文的一章。其中，
      • preamble.tex — 封面、摘要、指导教师、鸣谢……示例；
      • dogstick.tex — 插入图片示例；
      • appendix.tex — 插入表格示例；
      • src.tex — 插入程序代码示例；
  • example — 历年毕业论文范例，可供参考；

各文件的使用说明

swfcthesis.cls

  • 通常你不需要改动 swfcthesis.cls 文件，将 \documentclass{swfcthesis} 做为TeX文件的开始即可。
  • 写论文时，请参照 tutorial.tex 文件的开始部分。

tutorial — 一个可以当模板用的小教程

  • tutorial.tex — 源文件

    编译： 

    latexmk tutorial

    前提是你安装并配置好了 latexmk 。你可以参考我的 .latexmkrc 文件。

sample — 可以当模板用的小论文

  • sample.tex — 源文件
    • 写论文时，可以直截修改此文件的内容。当然也要根据需要修改 chapters 目录中的TeX文件。
    • 修改完毕，用 latexmk 编译。当然也可以直接用 xelatex 编译。以Linux平台为例： 
      xelatex --shell-escape sample && xelatex --shell-escape sample && xelatex --shell-escape sample

      此命令是将 sample.tex 文件连续编译了三次，以正确生成正文、目录、参考文献等内容。如果编译没出问题的话，当前目录下应该有新生成的PDF文件了（ sample.pdf ）。

  • chapters — 各章节的 tex 源文件
    • 写论文时，直截修改此目录中TeX文件的内容即可。
    • 可根据需要增加/删减章节数。当然 sample.tex 文件中相应的 \include{} 命令也要做相应的改动。
  • sample.bib — 参考文献
    • 参考此文件的格式来写参考文献；
    • 写论文时，
      1. 将 \addbibresource{sample.bib} 放在论文的 preamble 里面，也就是 \begin{document} 之前。
      2. 在想要生成参考文献的地方插入命令 \printbibliography[title=参考文献] 即可。

注意事项

  1. 不要随意改动你不理解的东西，以免造成不必要的麻烦。
  2. 建议用 Emacs + AucTeX 来编辑TeX文件。
    • 计信学院D215机房的Linux平台安装了完整的LaTeX和Emacs可供使用；
    • 计信学院 cs2/cs3 服务器上也安装了完整的LaTeX和Emacs可供使用；
      • 具体使用方法可以参考教程；
