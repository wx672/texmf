文件清单

本模版的最新版本可以从 https://github.com/wx672/texmf/tree/master/doc/latex/swfu/swfcthesis/tutorial 下载。 模版目录中包含如下内容：

  • swfuthesis.cls — 是论文模版的class文件，它决定了论文的排版格式
  • readme.html — 本说明文件
  • tutorial — 小教程，同时也是一个标准模板。其中包括，
    • tutorial.tex — 源文件
    • tutorial.pdf — 编译好的PDF
    • tutorial.bib — 参考文献
    • tutorial.mkv — 20分钟视频教程
  • sample — 完整的论文实例
    • sample.tex — TeX源文件
    • sample.pdf — 编译完成的PDF文件
    • sample.bib — 参考文献实例
    • chapters — 章节实例。此目录中的每个TeX文件对应论文的一章。其中，
      • preamble.tex — 封面、摘要、指导教师、鸣谢……示例
      • dogstick.tex — 插入图片示例
      • appendix.tex — 插入表格示例
      • src.tex — 插入程序代码示例
  • example — 历年毕业论文范例，可供参考

各文件的使用说明

swfuthesis.cls

  • 通常你不需要改动 swfuthesis.cls 文件，将 \documentclass{swfuthesis} 做为TeX文件的开始即可。
  • 写论文时，请参照 tutorial.tex 文件的开始部分。

tutorial — 一个可以当模板用的小教程

  • tutorial.tex — 源文件

    编译： 

    latexmk tutorial

    前提是你安装并配置好了 latexmk 。你可以参考我的 .latexmkrc 文件。

注意事项

  1. 不要随意改动你不理解的东西，以免造成不必要的麻烦。
  2. 建议用 Emacs + AucTeX 来编辑TeX文件。
  3. 如有任何问题或建议，欢迎 email 联系。
