文件清单

本模版的最新版本可以从 github 下载。 模版目录中包含如下内容：

  • swfuthesis.cls — 是论文模版的class文件，它决定了论文的排版格式
  • readme.html — 本说明文件
  • tutorial — 小教程，同时也是一个标准模板。其中包括
    • tutorial.tex — 源文件
    • tutorial.pdf — 编译好的PDF
    • tutorial.bib — 参考文献
    • tutorial.mkv — 20分钟视频教程

各文件的使用说明

swfuthesis.cls

  • 通常你不需要改动 swfuthesis.cls 文件，将 \documentclass{swfuthesis} 做为TeX文件的开始即可。
  • 写论文时，请参照 tutorial.tex 文件的开始部分。

tutorial — 一个可以当模板用的小教程

  • tutorial.tex — 源文件

    编译： 

    latexmk tutorial

    前提是你安装并配置好了 latexmk 。你可以参考我的 .latexmkrc 文件。

注意事项

  1. 不要随意改动你不理解的东西，以免造成不必要的麻烦。
  2. 建议用 Emacs + AucTeX 来编辑TeX文件。
  3. 如有任何问题或建议，欢迎 email 联系。
